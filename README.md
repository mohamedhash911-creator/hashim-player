## ✨ المميزات

- 📂 رفع فيديو من جهازك وتشغيله مباشرة
- ⏮️ / ⏭️ تخطي 10 ثوانٍ للأمام أو الخلف
- 🔄 تدوير الفيديو بزاوية 90°
- 🎬 وضع السينما (ملء الشاشة الحقيقي بدون حدود)
- 💜 خلفية بنفسجية ساحرة
- 🔊 تحكم بالصوت (رفع / خفض)
- ☀️ تحكم بإضاءة الفيديو (زيادة / إنقاص السطوع)
- 🖋️ توقيع "محمد هاشم" بطابع أنيق

---

## 🚀 كيفية التشغيل

1. حمّل الملف `index.html`
2. افتحه في أي متصفح حديث
3. اضغط على زر **📂 يلا نتفرج** واختر فيديو
4. استمتع بالتجربة التفاعلية 🎥

> لا يتطلب أي خادم – يعمل محلياً بالكامل

---

## 🛠️ التقنيات المستخدمة

- HTML5 (فيديو، إدخال ملفات)
- CSS3 (تدرجات، تأثيرات، وضع السينما)
- JavaScript (التحكم بالفيديو، الفلاتر، التدوير)

---

## 📝 ملاحظات

- وضع السينما يخفي العنوان لتجربة غامرة.
- تدوير الفيديو لا يؤثر على عناصر التحكم.
- يمكن تعديل قيم السطوع والصوت بالنقر على الأزرار.
